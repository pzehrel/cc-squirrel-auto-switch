---
description: Configure cc-squirrel-auto-switch statusline
allowed-tools: Bash
---

Run the install script:

```bash
bash ${CLAUDE_PLUGIN_ROOT}/install.sh
```

To uninstall:

```bash
bash ${CLAUDE_PLUGIN_ROOT}/install.sh --remove
```

After install, restart Claude Code. Press `i` for Chinese, `Esc` for ABC.
